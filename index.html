👋 关于我

👋 About Me

你好！我是陈蓉杰（Rongjie Chen），目前是福建师范大学计算机与网络空间安全学院的网络空间安全专业硕士研究生（2025–至今），研究聚焦于机器遗忘 (Machine Unlearning) 与隐私保护技术。我对“被遗忘权”、模型安全以及隐私合规充满热情，致力于设计高效且可验证的机器学习遗忘算法，从根源上保障数据隐私。

Hello! I'm Rongjie Chen, a Master's student in Cybersecurity at Fujian Normal University (2025–present). My research focuses on Machine Unlearning & Privacy-Preserving Technologies. I am deeply passionate about the “right to be forgotten”, model security, and privacy compliance, striving to design efficient and verifiable machine unlearning algorithms to protect data privacy from the ground up.

我相信隐私是一项基本权利。通过机器遗忘框架、差分隐私等技术，我们能够在用户数据删除请求、模型更新治理等场景中建立负责任的AI系统，让机器学习真正尊重个体隐私。

I believe privacy is a fundamental right. By leveraging machine unlearning frameworks, differential privacy, and related techniques, we can build responsible AI systems that honor data deletion requests and model governance, ensuring machine learning truly respects individual privacy.

🧽 基于SISA的高效机器遗忘框架

🧽 Efficient Unlearning via SISA Framework

实现针对图像分类任务的近似遗忘机制，将数据集分片训练并支持快速样本遗忘，遗忘时间相比重训练降低85%以上，保持模型精度。

Implement an approximate unlearning mechanism for image classification using SISA; reduces unlearning time by >85% compared to retraining while maintaining model accuracy.

🔏 差分隐私下的遗忘验证协议

🔏 Unlearning Verification under Differential Privacy

设计隐私保护遗忘验证协议，允许第三方在不泄露原始模型信息的前提下验证特定数据是否被成功遗忘，平衡隐私与可审计性。

Design a privacy-preserving unlearning verification protocol that allows a third party to verify data deletion without exposing original model parameters, balancing privacy and auditability.

📊 遗忘鲁棒性评估基准

📊 Benchmark for Unlearning Robustness

构建针对机器遗忘算法的攻击评估工具，度量遗忘后模型对成员推理攻击和遗忘痕迹提取的防御能力，推动安全遗忘标准。

Build an evaluation toolkit for machine unlearning robustness, measuring defenses against membership inference attacks and unlearning trace extraction, promoting secure unlearning standards.
